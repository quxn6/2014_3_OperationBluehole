=== For All Users ===

Please import your DFGUI or NGUI plugin ahead of this package.

If you have questions or need support, please contact us via gcassets@gmail.com.


=== For NGUI Users ===

The example scene is suppose to run under 1024px * 768px resolution, change it in Player Settings when switched to Desktop or Web Player platforms.

Please import NGUI 3.4.9(or later) before you unpack “NGUI Example.unitypackage”.


=== For DFGUI Users ===

If you are using DFGUI and don’t have NGUI plugin in your project, do not import “NGUI Example.unitypackage”, or it will report missing script errors.
